| **Tópico Tratado** | **Detalhamento de Decisões** |
| :--- | :--- |
| Feedback da Visão 1.0 | Apresentação e aprovação das Seções 1.1 a 2.3 do documento de Visão. |
| Complementaridade | Ficou definido que o app deve ser apresentado como aliado/complemento ao jogo físico, e não um substituto total, preservando o valor da manipulação física em sala. |
| Complexidade Linguística | Discussão sobre conceitos de Alomorfia (múltiplas formas para o mesmo significado) e a necessidade de o sistema lidar com essa abstração. |
| Plataforma | Decisão por focar em uma aplicação mobile (App), visando o comportamento da geração atual e facilidade de acesso rápido. |
| Acessibilidade | Alinhamento de que recursos para deficientes visuais serão tratados como desdobramentos futuros, não compondo o MVP inicial. |
